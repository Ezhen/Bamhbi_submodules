# NECCTON — BAMHBI (FABM submodules)

This documentation describes the BAMHBI FABM submodules contained in this repository.

Use the left navigation to browse:
- **Tracers / diagnostics / parameters** (registries from `fortran/Databases/`)
- **Process blocks** (one page per BAMHBI module)
- **Build & run** (how to compile as part of a FABM build, and how to use the example YAML configs)

> Tip: The process-block pages are written as “hats”: they define what each routine *reads*, *updates*, and *exposes* (rates/diagnostics), plus the parameters it consumes.
